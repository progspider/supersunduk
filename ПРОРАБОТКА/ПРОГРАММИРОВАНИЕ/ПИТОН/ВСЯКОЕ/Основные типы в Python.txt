Основные типы:

  str  'Bogdan'               Строки
  int  10                     Целые числа
 bool  True                   Логический тип
 list  [1, 2, 3]              Список
 dict  {'min': 5, 'max': 8}   Словарь


        str  'Bogdan'
         ↑      ↑
 имя класса    содержимое строки


Строки: строки записывают между одинарными (или двойными) ковычками.

Целые числа: 0 и отрицательные числа это тоже целое число.

Логический тип: есть только два значения: True и False.

Список: можно сохранять элементы разных типов, элементы разделены запятой, пробел после открывающей (и перед закрывающей) квадратной скобкой СТАВИТЬ НЕ НУЖНО, а после запятой ставить пробел.

Словарь: состоит из элементов пар КЛЮЧ: значение, названия ключей должны находиться между одинарными (или двойными) ковычками, и в словарях и в списках может быть любое количество элементов.
