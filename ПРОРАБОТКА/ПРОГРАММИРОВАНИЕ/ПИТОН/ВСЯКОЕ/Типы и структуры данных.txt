В PYTHON ОТСУТСТВУЮТ ПРИМИТИВНЫЕ ТИПЫ

В PYTHON СУЩЕСТВУЮТ ИЗМЕНЯЕМЫЕ И НЕИЗМЕНЯЕМЫЕ ОБЪЕКТЫ
                        ↑              ↑
                     Mutable         Immutable
       (Произносится: Мъютабл)        (Произносится: Имъютабл)
(Часто говорят: Мутировать объект)

НЕИЗМЕНЯЕМЫЕ ОБЪЕКТЫ: string (строка, str), boolean (логический, bool), integer (целое число, int), float (число с десятичной точкой), tuple (кортеж, tuple), None (ничто, NoneType), и.т.д.

ИЗМЕНЯЕМЫЕ ОБЪЕКТЫ: list (список, list), dictionary (словарь, dict), set (набор, set), user-defined classes (пользовательские классы).
