Python игнорирует (НЕ ВЫПОЛНЯЕТ) текст написанный как КОММЕНТАРИЙ.

Пример кода:

# Комментарий

# Многострочный
# комментарий

print(10)  # Код с комментарием


Комментарии бывают:

 - однострочные
 - многострочные
 - на строках с кодом

Бывают ещё комментарии которые находятся между тройными одинарными, либо тройными двойными ковычками, но такие комментарии обычно используют для описания функций.

ВАЖНЫЙ МОМЕНТ: не нужно добавлять комментарий который описывает то что выполняет данный код. Код должен быть читабельным и понятным другим пользователям. С помощью комментария можно описать цель кода, либо как его правильно использовать.
