В некоторых языках программирования для разделения блоков кода используются скобки. В Python же для этих целей используется отступы. А именно отступы сформированные на основании пробелов.

ОТСТУПЫ ИГРАЮТ ВАЖНУЮ РОЛЬ

Обычно отступы формируются с помощью 4-х пробелов.

# Корректный код
def print_name_function(name):
    print(name)


# Некорректный код
def print_name_function(name):
print(name)

Возникает ошибка: IndentationError: expected an indented block (ожидается блок кода с отступом). В теле функции должна быть хотя бы одна строчка кода, а её нет, потому что print(name) написан без отступов, поэтому и возникает ошибка.


# Корректный код
def my_name(name):
    print(name)

my_name('Ruslan')


# Некорректный код
def my_name(name):
    print(name)

    my_name('Ruslan')

Во втором примере отступ не нужен, в терминале мы ничего не увидим потому что вызова функции НЕТ (так как my_name('Ruslan') находится в теле функции).
