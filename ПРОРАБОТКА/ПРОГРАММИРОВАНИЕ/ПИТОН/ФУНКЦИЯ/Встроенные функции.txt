Вызов встроенной функции ( пример ► print("Hello World!") ):
    1. Пишется название встроенной функции, в данном случае это print.
	2. Далее пара круглых скобок, круглые скобки означает вызов функции.
	3. Внутри скобок пишем аргументы функции, в данном случае это значение типа str.

  ФУНКЦИЯ
  
  def my_fn(a, b):
      a = a + 1
	  c = a + b
	  return c

С помощью ключевого слова def можно создавать свои функции. У каждой функции есть имя (my_fn), кроме имени у функции есть тело, это набор строк которые выполняются при вызове той или иной функции, кроме тела функции у функции есть параметры, у функции есть строка с ключевым словом return, это означает что функция возвращает определенный результат.

Когда мы вызываем функцию, она выполняет определенные действия, функция вернет результат, этот результат можно присвоить определенной переменной.

Встроенные функции (тут не все, их много)

print()  type()  id()  len()  sum()  input()  round()  min()  max()  int()  str()  bool()

Если в качестве аргумента передать название встроенной функции, в функции print, то выводится сообщение о том что данная (которую вводили) функция ЯВЛЯЕТСЯ ВСТРОЕННОЙ (ниже примеры).

примеры:
>>> print(len)
<built-in function len>
>>> print(sum)
<built-in function sum>
>>> print(print)
<built-in function print>

ВСТРОЕННАЯ ФУНКЦИЯ INPUT

name = input("Enter your name: ")
print(name)

На заметку: то значение которое вводит пользователь, это всегда строка (значение типа str). Если вводятся числа, и нужно работать с числами, то нужно строку конвертировать в числовой тип (использовать функцию int() для целых чисел, float() для чисел с плавающей точкой).
