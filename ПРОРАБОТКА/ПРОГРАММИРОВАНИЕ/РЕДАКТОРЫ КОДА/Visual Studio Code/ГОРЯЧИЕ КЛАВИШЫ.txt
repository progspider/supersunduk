shift + Alt + КНОПКА_ВНИЗ  -  скопировать строку.
Alt зажать, нажимать вверх или вниз  -  ПЕРЕМЕЩЕНИЕ СТРОКИ

ctrl + shift + K  -  удалить строку.

ctrl + D  -  БЫСТРО ВЫДЕЛИТЬ ТЕКУЩЕЕ НАЗВАНИЕ (например имя переменной)

ctrl зажать, нажимать вправо либо влево  -  ПЕРЕМЕЩЕНИЕ между словами в строке

ЗАКОММЕНТИРОВАТЬ БЛОК КОДА: Блок кода выделить, нажать: ctrl + / (если снова нажать то раскомментировать)
