3 способа установки модулей в Python
============================================================================
1-ый способ: использовать pip, пример: pip install html2text --user
============================================================================
2-ой способ: установка из самого Питона:
---------------------------------------------------------------------------
import pip

pip.main(["install", "html2text", "--user"])
