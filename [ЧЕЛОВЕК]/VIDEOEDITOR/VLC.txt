5 СЕКРЕТОВ VLC Media Player

    1. Просмотр ТВ на компьютере: В поисковике, вводим: "скачать плейлист тв m3u", с любого найденного сайта скачать файл с расширением m3u, по файлу правой кнопкой мыши >> Добавить в плейлист VLC. Если теперь в плеере включить отображение плейлиста, то появиться список всех доступных телеканалов. Этот файл можно редактировать в любом текстовом редакторе. Некоторые каналы могут не работать по причине того что изменилась ссылка, тут можно либо искать свежий файл, либо скачать самообновляющийся плейлист.
	
	2. ЗАПИСЬ ФРАГМЕНТА или ОБРЕЗКА ВИДЕО: Во время воспроизведения видео или аудио - Воспроизведение >> Запись (захват видео начнется, но в плеере это никак не будет отображаться). Чтобы закончить запись повторяем те же самые действия. Файл с записанным фрагментом можно найти в стандартной папке "Видео" (для аудио это папка "Музыка"). Чтобы упростить запись можно вынести соответсвующую кнопку на панель упраления: Вид >> Доп. элементы управления, либо Инструменты >> Настройка интерфейса... >> Перетягиваем нужную кнопку на желаемое место.
	
	3. ЗАПИСЬ ЭКРАНА: Медиа >> Открыть устройство захвата... >> Режим захвата, выбираем: "Экран", указать желаемую частоту кадров, кнопка "Воспроизвести", нажать стрелку и выбрать "Конвертировать", выбрать подходящий профиль, выбрать место сохранения >> Нажать кнопку "Начать". Чтобы остановить запись: кнопка "Остановить" на панели управления проигрывателя.
	
	4. Поворот / отзеркаливание видео: Инструменты >> Эффекты и фильтры >> вкладка "Видеоэффекты" >> вкладка "Геометрия" >> Поставить галочку "Поворот" и указать желаемый угол поворота (любое просматриваемое видео будет повёрнуто). Затем "Медиа" >> "Конвертировать/сохранить..." >> Во вкладке "Файл", кнопка "Добавить..." >> Выбрать видео которое нужно повернуть >> кнопка "Конвертировать/сохранить" >> кнопка с гаечным ключём (редактировать профиль) на неё нажать >> вкладка "Видеокодек", и далее вкладка "Фильтры" >> Выбираем "Rotate video filter", кнопка "сохранить" >> указываем расположение будующего файла >> кнопка "начать". Фильтр поворота нужно отключить (настройка в самом начале).
	
	5. Конвертация видео в другой формат: Медиа >> Конвертировать/сохранить... >> Во вкладке "Файл", кнопка "Добавить..." >> Выбираем файл который хотим конвертировать >> Кнопка "Конвертировать/сохранить", стрелочка, и выбрать "Конвертировать" >> Выбрать подходящий профиль (если нужно можно настроить) >> Указать конечный файл >> кнопка "Начать".
